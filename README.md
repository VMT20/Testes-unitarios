## Aprendizados

Durante este laboratório, aprendi na prática como utilizar o Git e o GitHub para versionamento de código. Algumas das etapas que realizei foram:
- Criação e configuração de um repositório local e remoto.
- Uso do `git add` e `git commit` para salvar o histórico.
- Criação de branches (GitHub Flow).
- Em breve: Resolução de conflitos!
